Standard Shell Scheme

Basic Booth Design is the basic requirement for all exhibitions in the worldwide. To build the standard stand, we used simple design and light material.


Upgraded Shell Scheme


Upgraded Shell Scheme is a design whereby timber construction can be blend with shell scheme material. The design can be inexpensive rather than using all timber and metal construction. The size can shape from 2M x 2M minimum and up to maximum size, that customer required.


Pop-up System


The Pop-Up display system is fully magnetic and can be reused over and over. Eyes catching seamless large format illustration and photos make this pop-up display system attractive. Using the very latest ‘smart magnetic’ technology to ensure fast installation and easy graphic positioning.This Pop-up Display system adds high impact to any presentation or exhibition stand.

Organizer Requirement


Usually, for Organizer Requirement we put closely with basic requirement. There are some requirements that need to do to make the show in perfect condition and we will use shell scheme structure to construct these entire requirement.
